✅ ICÔNES GÉNÉRÉES AUTOMATIQUEMENT

Tous les fichiers PNG de ce dossier sont générés automatiquement depuis favicon.svg.

Pour régénérer :
  node scripts/generate-favicons.js

Formats :
  - icon16.png  (16x16)
  - icon48.png  (48x48)  
  - icon128.png (128x128)
  - icon512.png (512x512)

⚠️ NE PAS éditer manuellement. Modifiez favicon.svg puis relancez le script.
