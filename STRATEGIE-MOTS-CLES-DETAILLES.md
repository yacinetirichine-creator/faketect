## 🎯 Keyword Implementation Strategy

### On-Page Optimization Template

```
🎯 PRIMARY KEYWORD: "détecter image IA"
📌 SECONDARY: "détection images générées par IA"
🔗 LONG-TAIL: "comment détecter une image générée par IA", "vérifier si image IA"

ELEMENT | IMPLEMENTATION | EXAMPLE
---------|---|---
Title | Include primary KW, power words | "Détecter Images IA: Guide Complet 2025 + Outils"
Meta Description | KW + CTA (155 chars) | "Découvrez comment détecter images générées par IA. Guide pratique + outils gratuits pour vérifier authenticité photos."
H1 | Primary KW at start | "Détecter Images IA: Techniques & Outils Efficaces"
H2s | Include secondary keywords | "Comment fonctionne la détection d'images IA?"
First 100 words | Natural KW placement (2-3%) | Include KW in opening paragraph
Internal links | Link to related articles | "Lire aussi: [Related Article]"
Image alt text | Keyword variation | "Exemple image IA détecté par FakeTect"
URL slug | Keyword-based | "/blog/detecter-image-ia-2025"
```

### Keyword Density Guidelines

```
✅ CORRECT (Natural, Helpful):
- Primary KW: 0.5-1% of content
- Secondary KW: 0.3-0.7%
- Related terms: 1-2%
- Total keyword coverage: ~3-5%

❌ WRONG (Keyword Stuffing):
- Forcing keywords
- Unnatural phrasing
- Over-optimization
- Hidden text/keywords

🎯 BEST PRACTICE:
- Write for humans first
- Keywords come naturally
- Focus on synonym variation
- Use LSI keywords
```
